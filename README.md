# 수능만만 듣기 스크립트 추출기

수능만만 영어 듣기 모의고사 PDF에서 **한국인 성우 스크립트**(문항 지시문)와 **외국인 성우 스크립트**(W:/M: 지문)를 추출해 워드·엑셀·텍스트로 내보내는 웹 도구입니다.

## 사용 방법

1. **PDF 업로드** – 수능만만 듣기 PDF 선택
2. **스크립트 유형 선택** – 한국인 성우(지시문) / 외국인 성우(W:, M: 지문)
3. **스크립트 추출 시작** – PDF 분석 후 단원별 문항 데이터 추출
4. **폰트 크기** – 기본 15px, 미리보기로 확인 후 적용
5. **파일 다운로드** – 워드(.doc), 엑셀(.xlsx), 텍스트(.txt) 선택

파일명: `업로드파일명_YYYYMMDD_HHMMSS.확장자`

## 로컬 실행

- `index.html`을 브라우저에서 열거나, 로컬 웹 서버로 루트 디렉터리 제공 후 접속  
- 예: `npx serve .` 또는 `python -m http.server 8080`

## GitHub Pages 배포

1. 이 저장소를 GitHub에 푸시
2. **Settings → Pages**에서 Source를 **GitHub Actions** 또는 **main 브랜치 / (root)** 로 설정
3. `config.json`의 `deploy.baseUrl`은 서브경로 배포 시에만 설정 (예: `/english-listening-extract/`)

## 구성 파일

- `config.json` – 앱 이름, 기본 폰트 크기, 배포 설정
- `index.html` – 단일 페이지 UI
- `js/app.js` – PDF 파싱, 추출, 내보내기 로직

## 지원 PDF 구조

- **표지 → 구성/특징 → 목차 → 간지(교재명/단원명) → 단원/문제** 순서
- 단원/문제 페이지: 단원번호, 단원명, 듣기 안내 지시문, 문항 번호, 지시문 또는 W:/M: 지문
